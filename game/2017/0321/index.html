Burning D-man and Gopher

このゲームは、traP内で行われた3時間gamejamで制作したものです。
テーマ「着火と鎮火」
今回も時間内に間に合わなかった(-_-;)（3時間 + 1時間）
Twitterタグ #traP3jam

遊び方

すべてのGopherくんを燃やし尽くそう！
Gopherくんを燃やし残ったり、誤ってD言語くんを燃やしてしまったらGameOver。
制限時間内に着火と鎮火をするべき爆弾を見極めよう！
画面をクリックするとゲームがスタートするよ。

操作方法

  • 爆弾を左クリック ... 爆弾を着火
  • 爆弾を右クリック ... 爆弾を鎮火
  • GameOver画面でtボタン ... ツイート
  • GameOver画面でEnterキー ... リスタート

※画面をクリックするとゲームがスタートするよ。

※スマートフォンには対応していないのでPCでお遊びください。
